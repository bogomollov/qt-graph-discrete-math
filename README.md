<h1>Qt Graph Discrete Math</h1>
<p align="center">Академическое приложение по дискретной математики для работы с графами</p>

## Начало работы

1. Клонировать репозиторий
```bash
git clone https://github.com/bogomollov/qt-graph-discrete-math.git
```

2. Открыть проект в Qt Creator

3. Запустить нужный тип сборки

## Стек технологий

- C++
- Qt 6.11.0 (QWidget, QPainter)
