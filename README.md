## Начало работы

1. Клонировать репозиторий
```bash
git clone https://github.com/bogomollov/qt-graph-discrete-math.git
```

2. Активировать githooks
```bash
git config core.hooksPath .githooks
```

3. Открыть проект в Qt Creator

4. Запустить нужный тип сборки

## Стек технологий

- C++
- Qt 6.11.0 (Widgets, Graphics View)
